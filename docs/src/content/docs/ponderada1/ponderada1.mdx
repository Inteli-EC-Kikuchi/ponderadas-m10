# Ponderada 1

Implementação de uma API para ToDo List utilizando Go com o framework chi de forma síncrona. O projeto utiliza requisições HTTP com autenticação básica e verificação JWT para verificar operações relacionadas ao usuário.

## Features
- Operações de CRUD (Create, Read, Update and Delete) para as tarefas.
- Registro de usuários
- Autorização para verificação de JWT
- Containerização da aplicação com Docker

## Execução

Para executar o projeto, siga os seguintes passos:

import { Steps } from '@astrojs/starlight/components';

<Steps>

1. Clone o repositório:

    ```bash
    git clone https://github.com/Inteli-EC-Kikuchi/ponderadas-m10.git
    ```

2. Entre na pasta do projeto:

    ```bash
    cd ponderadas-m10/ponderada1/go
    ```

3. Execute o comando:

    ```bash
    docker-compose up
    ```
